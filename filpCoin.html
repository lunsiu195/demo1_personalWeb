Filp Coin

Website and Cloud Secutity

Source code / Project Document

Year: 2023

Abstract

Our coin flipping website using Apache web server will allow users to initiate coin tosses. Our website will consist of a clean and simple interface with a button to generate the coin flip. Once the user selects the coin flip button, the server will generate either heads or tails to determine the winner. This result will be showcased to the user and if the user requests, the process can be repeated multiple times. Along with showcasing the result, our website can also track the number of wins and show the statistics for both heads and tails. Throughout this report, we will showcase how Apache is a reliable web server and can provide the necessary protective measures to host our coin flip website to users.
